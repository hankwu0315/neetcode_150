# 0002. Add Two Numbers

- 難度：Medium
- LeetCode：https://leetcode.com/problems/add-two-numbers/

## 題目描述（中文）

兩個非負整數以**逆序**儲存在串列中，每節點為一位數。回傳兩數相加的結果，同樣以逆序串列表示。

## 範例 / 限制

- `[2,4,3] + [5,6,4]` → `[7,0,8]`（342 + 465 = 807）
- `[0] + [0]` → `[0]`
- `[9,9,9,9,9,9,9] + [9,9,9,9]` → `[8,9,9,9,0,0,0,1]`

## 解法

### 解法（最佳解）：dummy + 進位

同步走兩串列，逐位相加加上進位，建新節點。
```
carry = 0
while a or b or carry:
    s = (a.val if a) + (b.val if b) + carry
    carry, digit = divmod(s, 10)
    tail.next = ListNode(digit)
    ...
```
- 時間：O(max(m, n))
- 空間：O(max(m, n))

## 範例追蹤

`[2,4,3] + [5,6,4]`：
- 2+5=7, c=0 → 7
- 4+6=10, c=1 → 0
- 3+4+1=8, c=0 → 8
- 結果 `[7,0,8]` ✓

## 關鍵點 / 易錯陷阱

- **迴圈條件需含 `carry`**：否則最後一位的進位會丟失（如 `[5]+[5]=[0,1]`）。
- 串列長度不同時要把缺位視為 0。
- 不可改 input；建立新節點。
