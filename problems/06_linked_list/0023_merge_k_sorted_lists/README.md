# 0023. Merge K Sorted Lists

- 難度：Hard
- LeetCode：https://leetcode.com/problems/merge-k-sorted-lists/

## 題目描述（中文）

給定 `k` 個升序鏈結串列，合併為一個升序串列回傳。設總節點數 `N`。

## 範例 / 限制

- `[[1,4,5],[1,3,4],[2,6]]` → `[1,1,2,3,4,4,5,6]`
- `[]` → `[]`
- `[[]]` → `[]`

## 解法

### 解法 1：暴力收集再排序

把所有值取出排序後重新建串列。
- 時間：O(N log N)
- 空間：O(N)

### 解法 2：兩兩合併（分治）（最佳解之一）

把 `k` 個串列兩兩合併，類似合併排序。每輪合併後串列數減半。
- 時間：O(N log k)
- 空間：O(1)（迭代合併）

### 解法 3：最小堆（最佳解之一）

把每個串列頭丟入 min-heap，每次彈出最小、再把其 next 丟入。
- 時間：O(N log k)
- 空間：O(k)

## 範例追蹤（解法 2）

`[L1, L2, L3, L4]`：
- 第一輪：合併 L1+L2、L3+L4 → 2 串。
- 第二輪：合併剩下兩串 → 1 串。

## 關鍵點 / 易錯陷阱

- 處理空 input 與含空串列的情況。
- 解法 3 的 heap 比較需 tie-breaker：Python 把 `(val, idx, node)` 入堆，避免比較 `node` 物件。
- 兩兩合併採「步長」寫法：`for i in range(0, k, step*2): merge(lists[i], lists[i+step])`。
