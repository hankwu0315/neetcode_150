# 0143. Reorder List

- 難度：Medium
- LeetCode：https://leetcode.com/problems/reorder-list/

## 題目描述（中文）

`L0 → L1 → … → Ln-1` 重排為 `L0 → Ln-1 → L1 → Ln-2 → L2 → Ln-3 → …`，**就地修改**，不能改變節點值。

## 範例 / 限制

- `[1,2,3,4]` → `[1,4,2,3]`
- `[1,2,3,4,5]` → `[1,5,2,4,3]`

## 解法

### 解法（最佳解）：找中點 + 反轉後半 + 交錯合併

1. **找中點**：快慢指標，`slow` 走到中點。
2. **切斷並反轉後半**：`second = slow.next; slow.next = None`；對 `second` 反轉。
3. **交錯合併**：兩串列從前往後一節一節接。

- 時間：O(n)
- 空間：O(1)

### 解法（暴力）：陣列 + 雙指標

把所有節點放入陣列，雙指標從首尾交錯接回。
- 時間：O(n)
- 空間：O(n)

## 範例追蹤

`[1,2,3,4,5]`：
- 中點：`slow=3`。前半 `1->2->3`，後半（反轉前 `4->5`）反轉 → `5->4`。
- 合併：`1->5`、`2->4`、`3` → `1->5->2->4->3`。

## 關鍵點 / 易錯陷阱

- 找中點寫法決定前半長度：用 `while fast.next and fast.next.next` 讓奇數時 slow 在正中（前半多 1）；偶數時 slow 在「左中」。本題兩種長度配比都可，但**切斷後 slow.next 必須設 None** 防環。
- 合併時要先暫存 `next`，避免覆寫。
