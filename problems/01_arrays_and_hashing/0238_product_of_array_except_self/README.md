# 238. Product of Array Except Self

**Difficulty**: Medium  **URL**: https://leetcode.com/problems/product-of-array-except-self/

## 題目描述（中文）

給定整數陣列 `nums`，返回一個新陣列 `answer`，其中 `answer[i]` 等於 `nums` 中**除了 `nums[i]` 以外**所有元素的乘積。  
題目保證任意前綴或後綴乘積都在 32-bit 整數範圍內，且**不可使用除法**。

**進階**：要求時間 O(n)，且輸出陣列不計入額外空間（即 O(1) 額外空間）。

## 範例 / 限制

**範例 1**
```
Input:  nums = [1,2,3,4]
Output: [24,12,8,6]
```

**範例 2**
```
Input:  nums = [-1,1,0,-3,3]
Output: [0,0,9,0,0]
```

**限制**
- `2 <= nums.length <= 10⁵`
- `-30 <= nums[i] <= 30`
- 不可使用除法
- 進階：O(1) 額外空間（不含輸出）

## 解法

### 解法一：暴力法（嵌套迴圈）
對每個 `i`，重新算其餘元素乘積。
- **時間複雜度**：O(n²)
- **空間複雜度**：O(1)

### 解法二：左右前綴乘積（兩個輔助陣列）
`left[i]` = nums[0..i-1] 的乘積；`right[i]` = nums[i+1..n-1] 的乘積。  
則 `answer[i] = left[i] * right[i]`。
- **時間複雜度**：O(n)
- **空間複雜度**：O(n)

### 最佳解：原地兩遍掃描

**核心思路**：先在 `answer` 中放「左前綴」，再用一個變數 `right` 累乘右後綴並就地相乘。

1. 第一遍由左到右：`answer[i]` = `nums[0..i-1]` 的乘積（`answer[0] = 1`）。
2. 第二遍由右到左：維護 `right = 1`，每步 `answer[i] *= right`，再 `right *= nums[i]`。

如此 `answer[i] = (左前綴) × (右後綴) = 除自身外的乘積`。

- **時間複雜度**：O(n)
- **空間複雜度**：O(1)（不計輸出）

## 範例追蹤（最佳解）

```
nums = [1,2,3,4]

第一遍（左前綴，answer[0]=1）：
  i=1: answer[1] = answer[0]*nums[0] = 1*1 = 1
  i=2: answer[2] = answer[1]*nums[1] = 1*2 = 2
  i=3: answer[3] = answer[2]*nums[2] = 2*3 = 6
  answer = [1, 1, 2, 6]

第二遍（右後綴，right=1）：
  i=3: answer[3] = 6*1 = 6,  right = 1*4 = 4
  i=2: answer[2] = 2*4 = 8,  right = 4*3 = 12
  i=1: answer[1] = 1*12 = 12, right = 12*2 = 24
  i=0: answer[0] = 1*24 = 24, right = 24*1

結果：[24, 12, 8, 6] ✓
```

## 關鍵點 / 易錯陷阱

- 不可使用除法（即使原陣列無 0）。
- 含 0 的特殊情況：兩個 0 → 全部為 0；只有一個 0 → 該位置為其他元素積，其餘位置為 0。
- 第二遍**必須由右往左**。
- `answer` 陣列不算額外空間，這是進階要求的關鍵。
