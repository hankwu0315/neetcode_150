# 11. Container With Most Water

**Difficulty**: Medium  **URL**: https://leetcode.com/problems/container-with-most-water/

## 題目描述（中文）

給定整數陣列 `height`，其中 `height[i]` 表示 x = i 處垂直線的高度。  
找出兩條線，使它們與 x 軸圍成的容器**儲水量最大**，回傳該最大儲水量。

儲水量 = `min(height[i], height[j]) * (j - i)`。

## 範例 / 限制

**範例 1**
```
Input:  height = [1,8,6,2,5,4,8,3,7]
Output: 49   (i=1, j=8 → min(8,7)*7 = 49)
```

**範例 2**
```
Input:  height = [1,1]
Output: 1
```

**限制**
- `2 <= height.length <= 10⁵`
- `0 <= height[i] <= 10⁴`

## 解法

### 解法一：暴力雙重迴圈
列舉所有 `(i, j)`，計算容量取最大。
- **時間複雜度**：O(n²)
- **空間複雜度**：O(1)

### 最佳解：雙指標相向收斂

**核心思路**：

1. 設 `l = 0, r = n - 1`，初始寬度最大。
2. 計算 `area = min(h[l], h[r]) * (r - l)`，更新最大值。
3. **永遠移動較矮的一側**：
   - 較矮的一側決定容量，留住較高側仍可能與更高的對手配對；
   - 移動較高側只會讓寬度與最小高度都不利，必劣於目前。
4. 重複直到 `l >= r`。

每步排除一個指標，因此線性。

- **時間複雜度**：O(n)
- **空間複雜度**：O(1)

## 範例追蹤（最佳解）

```
height = [1,8,6,2,5,4,8,3,7], n=9

l=0(1), r=8(7): min=1, w=8, area=8       best=8
  h[l]<h[r] → l=1
l=1(8), r=8(7): min=7, w=7, area=49      best=49
  h[l]>h[r] → r=7
l=1(8), r=7(3): min=3, w=6, area=18
  h[l]>h[r] → r=6
l=1(8), r=6(8): min=8, w=5, area=40
  相等，可任選 → r=5
l=1(8), r=5(4): min=4, w=4, area=16 → r=4
l=1(8), r=4(5): min=5, w=3, area=15 → r=3
l=1(8), r=3(2): min=2, w=2, area=4  → r=2
l=1(8), r=2(6): min=6, w=1, area=6  → r=1 結束

best = 49 ✓
```

## 關鍵點 / 易錯陷阱

- 移動較矮的一側才有可能使最小高度上升。
- 若兩側相等，移動任一側皆可（不影響正確性）。
- 寬度 `r - l`，不是 `r - l + 1`。
- 高度為 0 的邊不必特殊處理；雙指標自然處理。
