# 15. 3Sum

**Difficulty**: Medium  **URL**: https://leetcode.com/problems/3sum/

## 題目描述（中文）

給定整數陣列 `nums`，找出所有**三元組** `[nums[i], nums[j], nums[k]]` 使得 `i != j != k` 且和為 `0`。  
回傳結果**不可包含重複的三元組**（順序不限）。

## 範例 / 限制

**範例 1**
```
Input:  nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
```

**範例 2**
```
Input:  nums = [0,1,1]
Output: []
```

**範例 3**
```
Input:  nums = [0,0,0]
Output: [[0,0,0]]
```

**限制**
- `3 <= nums.length <= 3000`
- `-10⁵ <= nums[i] <= 10⁵`

## 解法

### 解法一：暴力三層迴圈
列舉所有三元組並去重。
- **時間複雜度**：O(n³)
- **空間複雜度**：O(n)（去重）

### 最佳解：排序 + 固定一個 + 雙指標

**核心思路**：

1. 先對 `nums` 排序。
2. 固定第一個數 `nums[i]`：
   - 若 `nums[i] > 0`：後續和必為正，無解 → 結束。
   - 若 `i > 0 且 nums[i] == nums[i - 1]`：跳過避免重複。
3. 在區間 `[i+1, n-1]` 用雙指標 `l, r` 找 `nums[l] + nums[r] == -nums[i]`：
   - `total < 0` → `l += 1`；
   - `total > 0` → `r -= 1`；
   - `total == 0` → 收進結果，並**同時跳過 `l`/`r` 的重複值**。

排序帶來去重便利與雙指標的可行性。

- **時間複雜度**：O(n²)
- **空間複雜度**：O(1)（不計輸出與排序內部）

## 範例追蹤（最佳解）

```
nums = [-1,0,1,2,-1,-4] → 排序後 [-4,-1,-1,0,1,2]

i=0, nums[i]=-4: 找 sum=4
  l=1(-1), r=5(2): -1+2=1 < 4 → l=2
  l=2(-1), r=5(2): 1 < 4 → l=3
  l=3(0),  r=5(2): 2 < 4 → l=4
  l=4(1),  r=5(2): 3 < 4 → l=5 結束

i=1, nums[i]=-1: 找 sum=1
  l=2(-1), r=5(2): -1+2=1 ✓ → [-1,-1,2]，l=3, r=4
  l=3(0),  r=4(1): 0+1=1 ✓ → [-1,0,1]，l=4, r=3 結束

i=2, nums[i]=-1: 與 i=1 相同 → 跳過

i=3, nums[i]=0: 找 sum=0
  l=4(1), r=5(2): 3 > 0 → r=4 結束

i=4: nums[i]=1 > 0 → 結束

結果：[[-1,-1,2], [-1,0,1]] ✓
```

## 關鍵點 / 易錯陷阱

- **必須先排序**才能用雙指標 + 跳重複技巧。
- 跳重複的兩處：
  1. 外層固定 `i` 時：`if i > 0 and nums[i] == nums[i - 1]: continue`。
  2. 找到答案後內層：`while l < r and nums[l] == nums[l + 1]: l += 1`（同理 `r`）。
- 找到答案後 `l` 與 `r` 都要前進一步（避免無窮迴圈）。
- 提前剪枝：`nums[i] > 0` 時可結束。
