# 0778. Swim in Rising Water

- 難度：Hard
- LeetCode：https://leetcode.com/problems/swim-in-rising-water/

## 題目描述（中文）

n×n 高度地圖，水位 t 時刻可走到所有 ≤ t 的格子。從 (0,0) 到 (n-1,n-1) 所需的最短時間。

## 範例 / 限制

- `[[0,2],[1,3]]` → 3
- `[[0,1,2,3,4],[24,23,22,21,5],[12,13,14,15,16],[11,17,18,19,20],[10,9,8,7,6]]` → 16

## 解法

### 解法（Dijkstra 變種）

把「到達一格的成本」定義為「沿途最大高度」（min-max 路徑）。最小堆優先擴展當前最小最大高度的格子。

時間 O(n² log n)，空間 O(n²)。

## 關鍵點 / 易錯陷阱

- 鬆弛條件：`new_max = max(cur_max, grid[ni][nj])`，比較最壞高度而非總和。
