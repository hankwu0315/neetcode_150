# 0020. Valid Parentheses

- 難度：Easy
- LeetCode：https://leetcode.com/problems/valid-parentheses/

## 題目描述（中文）

給定一個只包含 `()`、`[]`、`{}` 三種括號字元的字串 `s`，判斷它是否為「有效的括號字串」。

有效需滿足：
1. 左括號必須以對應型別的右括號關閉。
2. 左括號必須以正確的順序關閉。
3. 每個右括號必須有對應的左括號。

## 範例 / 限制

- `s = "()"` → `true`
- `s = "()[]{}"` → `true`
- `s = "(]"` → `false`
- `s = "([)]"` → `false`
- `s = "{[]}"` → `true`
- 1 ≤ s.length ≤ 10^4，僅含 `()[]{}`

## 解法

### 解法 1：堆疊（Stack）— 最佳解

掃描每個字元：
- 若是左括號 → 推入堆疊。
- 若是右括號 → 檢查堆疊頂端是否為對應的左括號；不是或堆疊為空，立即回傳 `false`。
- 掃描結束後堆疊必須為空（沒有未關閉的左括號）。

用 `dict` 對應 `)→(`、`]→[`、`}→{`，程式碼簡潔。

- 時間：O(n)
- 空間：O(n)（堆疊最壞情況存所有左括號）

### 解法 2：反覆 replace（暴力替代）

不斷把 `()`、`[]`、`{}` 從字串中移除，直到不變；最後若為空字串即有效。

- 時間：O(n^2)（最壞每輪只刪一對）
- 空間：O(n)
- 雖然簡短但效能差，僅做為對照。

## 範例追蹤（最佳解）

`s = "{[]}"`

| 字元 | 動作 | 堆疊 |
|----|----|----|
| `{` | push | `['{']` |
| `[` | push | `['{', '[']` |
| `]` | 配對 `[` ✓ pop | `['{']` |
| `}` | 配對 `{` ✓ pop | `[]` |

結束時堆疊空 → `true`。

## 關鍵點 / 易錯陷阱

- 遇到右括號時若堆疊為空 → `false`（沒有對應左括號）。
- 結束時忘了檢查堆疊非空 → 對於 `"("` 會誤判為 `true`。
- 字典查表比 if/elif 巢狀更清晰。
