# 0004. Median of Two Sorted Arrays

- 難度：Hard
- LeetCode：https://leetcode.com/problems/median-of-two-sorted-arrays/

## 題目描述（中文）

給定兩個升序陣列 `nums1`、`nums2`（總長 `m + n`），求其合併後的中位數。要求 **O(log(min(m, n)))**。

## 範例 / 限制

- `nums1 = [1,3], nums2 = [2]` → `2.0`
- `nums1 = [1,2], nums2 = [3,4]` → `2.5`
- 0 ≤ m, n ≤ 1000；1 ≤ m + n ≤ 2000；保證 m + n ≥ 1。

## 解法

### 解法 1：合併排序 O(m + n)

將兩陣列合併（合併排序的 merge 步驟）後取中位數。常用且穩定，但不滿足 log 要求。

### 解法 2：二分切割（最佳解）

讓 `nums1` 為較短陣列。對其切割位置 `i ∈ [0, m]` 二分，對應 `nums2` 切割 `j = (m + n + 1) // 2 - i`。

切割後的左半 = `nums1[0..i)` ∪ `nums2[0..j)`，右半 = `nums1[i..)` ∪ `nums2[j..)`。
合法切割需滿足：
- `nums1[i-1] <= nums2[j]`
- `nums2[j-1] <= nums1[i]`

（缺項以 ±∞ 替代）

判斷與調整：
- 若 `nums1[i-1] > nums2[j]`：`i` 太大 → `r = i - 1`
- 否則：`l = i + 1`

合法後：
- 若 `(m + n)` 為奇數：中位數 = `max(nums1[i-1], nums2[j-1])`。
- 為偶數：中位數 = `(max(左) + min(右)) / 2`。

- 時間：O(log(min(m, n)))
- 空間：O(1)

## 範例追蹤（解法 2）

`nums1 = [1, 3], nums2 = [2]`，m=2, n=1 → 交換使 `nums1=[2]`（短）, `nums2=[1,3]`，m=1, n=2。
總長奇 → halfLen = (1+2+1)//2 = 2。
`l=0, r=1`：
- i=0 → j=2 → 左1=`-inf`, 右1=`nums1[0]=2`；左2=`nums2[1]=3`, 右2=`inf`。
  檢查 `nums2[j-1]=3 <= nums1[i]=2` ❌（3 > 2）→ i 太小 → l=1
- i=1 → j=1 → 左1=`nums1[0]=2`, 右1=`inf`；左2=`nums2[0]=1`, 右2=`nums2[1]=3`。
  `2<=3` ✓ 且 `1<=inf` ✓ → 合法。
  奇數 → 中位數 = `max(2, 1) = 2.0` ✓

## 關鍵點 / 易錯陷阱

- 必須對 **較短陣列** 二分，確保 `j` 不會越界。
- `j = (m + n + 1) // 2 - i`，加 1 是為了讓奇數時左邊多 1 個元素，方便取中位數。
- 邊界：`i == 0` 時 `nums1[i-1] = -inf`；`i == m` 時 `nums1[i] = +inf`；`j` 同理。
- 浮點輸出：奇數時也要回 `float`。
