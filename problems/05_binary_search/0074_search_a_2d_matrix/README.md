# 0074. Search a 2D Matrix

- 難度：Medium
- LeetCode：https://leetcode.com/problems/search-a-2d-matrix/

## 題目描述（中文）

給定 `m x n` 矩陣 `matrix`，滿足：
1. 每列從左到右遞增。
2. 每列第一個元素大於前一列最後元素（**整個矩陣攤平後是嚴格遞增**）。

判斷 `target` 是否存在。要求 O(log(m*n))。

## 範例 / 限制

- `matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3` → `true`
- 同上 matrix, `target = 13` → `false`
- 1 ≤ m, n ≤ 100；-10^4 ≤ matrix[i][j], target ≤ 10^4。

## 解法

### 解法 1：將二維視為一維二分搜尋（最佳解）

由於攤平後嚴格遞增，視為長度 `m*n` 的有序陣列做二分。
索引 `idx` 對應 `(idx / n, idx % n)`。

- 時間：O(log(m*n))
- 空間：O(1)

### 解法 2：兩次二分

先二分找出 `target` 可能在哪一列（用每列第一個元素 ≤ target），再對該列二分。
- 時間：O(log m + log n) = O(log(m*n))，與解法 1 等價。
- 空間：O(1)

### 解法 3：暴力線性

雙重迴圈，O(m*n)。對照用。

## 範例追蹤（解法 1）

`matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]]`，攤平視角：`[1,3,5,7,10,11,16,20,23,30,34,60]`。

`target = 3`：
- l=0, r=11, mid=5 → 11 > 3 → r=4
- l=0, r=4, mid=2 → 5 > 3 → r=1
- l=0, r=1, mid=0 → 1 < 3 → l=1
- l=1, r=1, mid=1 → 3 = 3 → 命中 ✓

## 關鍵點 / 易錯陷阱

- `mid` 轉二維座標：`row = mid // n, col = mid % n`，**用 `n`（行數）而非 `m`**。
- 兩次二分版本要小心邊界：選擇 row 時用「最後一個 first 元素 ≤ target」。可改為一維二分避免錯誤。
