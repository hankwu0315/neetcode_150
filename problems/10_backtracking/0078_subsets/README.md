# 0078. Subsets

- 難度：Medium
- LeetCode：https://leetcode.com/problems/subsets/

## 題目描述（中文）

給定整數陣列 `nums`（**元素互異**），回傳其所有子集（power set）。順序不限。

## 範例 / 限制

- `[1,2,3]` → 8 個子集 `[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]`
- 1 ≤ nums.length ≤ 10。

## 解法

### 解法（回溯）：選或不選

對每個位置 `i`，遞迴兩個分支：
1. 不選 `nums[i]`，往下走
2. 選 `nums[i]`，加入路徑後往下走，回溯時 pop

時間 O(N · 2^N)（每個子集都要複製進結果），空間 O(N)。

### 解法 2：迭代

從 `[[]]` 開始，對每個 `x`：把現有所有子集加上 `x` 放回結果。

## 範例追蹤

`[1,2]`：
- start `[[]]`
- 加 1：`[[],[1]]`
- 加 2：`[[],[1],[2],[1,2]]`

## 關鍵點 / 易錯陷阱

- 結果裡每個子集都要 deep copy（path 是共享的）。
