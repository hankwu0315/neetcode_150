# 0994. Rotting Oranges

- 難度：Medium
- LeetCode：https://leetcode.com/problems/rotting-oranges/

## 題目描述（中文）

`0`=空、`1`=新鮮、`2`=腐爛。每分鐘所有腐爛的橘子會把 4-鄰新鮮的同時感染。求最少分鐘讓所有新鮮腐爛。若無法回 -1。

## 範例 / 限制

- `[[2,1,1],[1,1,0],[0,1,1]]` → 4
- `[[2,1,1],[0,1,1],[1,0,1]]` → -1

## 解法

### 解法（多源 BFS）

把所有 `2` 推入佇列，記新鮮數 fresh。BFS 一層一層擴散：
- 每層擴散完 `mins+=1`，將相鄰 `1` 變 `2` 並加入佇列、fresh--。

最後若 `fresh > 0` 回 -1，否則回最後 `mins`。

注意：若起始無新鮮，回 0。

時間 O(m·n)，空間 O(m·n)。

## 關鍵點 / 易錯陷阱

- 回傳是「最後一輪有實際擴散」的計數；若初始 fresh=0 直接回 0。
