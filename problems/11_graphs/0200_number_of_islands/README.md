# 0200. Number of Islands

- 難度：Medium
- LeetCode：https://leetcode.com/problems/number-of-islands/

## 題目描述（中文）

m×n 網格 `'1'` 為陸地、`'0'` 為水。上下左右相連視為同一島，求島嶼數量。

## 範例 / 限制

- `[["1","1","0"],["1","0","0"],["0","0","1"]]` → 2
- 1 ≤ m, n ≤ 300。

## 解法

### 解法（DFS 標記）

掃每格，遇 `'1'` 就 dfs 把整片變 `'0'`，計數+1。
時間 O(m·n)，空間 O(m·n)。
